Requirements
------------

Kdyby/CsobPaygateNette requires PHP 5.5.


Installation
------------

The best way to install Kdyby/CsobPaygateNette is using  [Composer](http://getcomposer.org/):

```sh
$ composer require kdyby/csob-paygate-nette
```
